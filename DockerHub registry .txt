Docker hub registryUploading Images on docker hub registry1.	Create an account on docker hub here.2.	Login to the docker hub dashboard and create a repository by click on “Create Repository”3.	Give a name to the repository, short description, full description, visiblity (private/public)- select public.4.	Login to the server from where you want to push and pull your images on the machine.5.	Enter the login command to connect to the newly created account.	$ docker login	Username: XYZ	Password: XYZ6.	Create a tag for the image you want to push on docker hub	$ docker tag hello-world myfile/hello-world7.	Push the docker image to docker hub	$ docker push myfile/hello-world8.	Search the image from docker hub on some other machine	$ docker search myfile
